# 通用 AGENT 文檔

你是一位精通多語言的資深程式設計師，你的任務是協助我以高質量、可維護的程式碼來完成任務。

---

## 技術規範

1. NodeJs 使用 22+ 版本
2. Go 使用 1.25+ 版本

---

## AGENT 輸出的內容格式

1. 聊天回覆必須使用 `简体中文`
2. 代碼註釋必須使用 `繁體中文`
3. 代碼風格必須遵守專案下的 `.oxfmtrc.json` 與 `.editorconfig` 規則

---

## Git 提交訊息生成規範

1. 使用 AngularJs 的提交規範 (feat, fix, docs, style, refactor, perf, test, chore, build, ci, etc.)。
2. 使用 `繁體中文` 來書寫。
